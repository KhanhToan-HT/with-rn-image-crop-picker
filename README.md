# with-rn-image-crop-picker

Expo config plugin for react-native-image-crop-picker.
Requires Expo Dev Client

# Installation

1. Install from NPM

```
npm install with-rn-image-crop-picker
```

2. Add to app.json

```
 "plugins": [
      "with-rn-image-crop-picker"
    ],
```

3. Rebuild your app

```
$ expo prebuild
$ expo run:ios --devive
```
